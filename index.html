cuJSON: A Highly Parallel JSON Parser for GPUs [Paper][Code]

GPU, JSON, High-Performace Data Parsing

Ashkan Vedadi Gargary, Soroosh Safari Loaliyan, and Zhijia Zhao | Proceedings of the 31th International Conference on Architectural Support for Programming Languages and Operating System, 2026

cuJSON is a highly parallel JSON parser that achieves 2.5x speedup over the state-of-the-art GPU JSON parser, GJSON, and 3.6x speedup over the best CPU JSON parser.

April 2023 - April 2025

BFSBlitz: A Highly Parallel Graph System for Breadth-First Search [Paper]

Graph Processing, Parallel Programming

Sakib Fuad, Ashkan Vedadi Gargary | FCPC 2025

A parallel BFS system that adapts between TopDown and BottomUp traversal strategies. Achieved up to \textbf{34.6$\times$ speedup} over serial BFS via hybrid partitioning and dynamic traversal.

Oct 2024 - March 2025

Hyb-Stepping: Hybrid Stepping for Parallel Shortest Paths [Paper]

Graph Processing, Parallel Programming

Ashkan Vedadi Gargary, Sakib Fuad | FCPC 2025

A dynamic shortest-path algorithm combining delta* and rho-stepping for optimal parallel performance. Introduced graph classification via 4 structural metrics to automate the stepping method and parameter selection. Outperformed fixed-parameter baselines with up to 1.5x speedup on large-scale graphs.

Oct 2024 - March 2025
